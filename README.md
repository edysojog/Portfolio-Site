# Interactive Terminal Portfolio 

A personal portfolio site styled as an interactive Linux terminal. Visit it live at **[edysojog.github.io/Portfolio-Site](https://edysojog.github.io/Portfolio-Site)**.

## Features

- Interactive terminal emulator — type commands to navigate the site
- Autocomplete cycling with `Tab`
- Keyboard click sound effects
- Background music player (`music` command)
- Animated cursor follower (Bocchi)
- Snake game built into the terminal

## Commands

| Command | Description |
|---------|-------------|
| `aboutme` | Who I am |
| `projects` | My projects |
| `social` | Social links |
| `email` | My email address |
| `music` | Play / pause background music |
| `snake` | Play Snake |
| `history` | View command history |
| `sudo` | Try it... |
| `clear` | Clear the terminal |
| `exit` | Close the tab |

## Built With

- **HTML / CSS / Vanilla JavaScript** — no frameworks
- **Web Audio API** — keyboard click sounds
- **Font Awesome** — icons
- **Fantasque Sans Mono** — terminal font
